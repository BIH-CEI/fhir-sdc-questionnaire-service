Overview

This workshop environment demonstrates technical challenges and solutions for Patient-Reported Outcome Measures (PROMs) interoperability using FHIR SDC (Structured Data Capture).

PROMIS Exchange

Demonstrate working FHIR questionnaire exchange between sites with different configurations.

Version Conflicts

Explore scoring algorithm version mismatches and their impact on patient safety.

Terminology Services

Enhanced LOINC terminology proxy with ordinal values and multilingual support.

Quick Start

Prerequisites: Docker and Docker Compose installed

# Clone repository
git clone https://github.com/nicktiberi/fhir-sdc-questionnaire-service.git
cd fhir-sdc-questionnaire-service

# Start workshop environment
docker compose -f docker-compose.workshop.yml up -d

# Wait 2-3 minutes for services to initialize
docker compose -f docker-compose.workshop.yml logs -f

# Access Demo UI
open http://localhost:8000
Note: HAPI FHIR can take 2-3 minutes to fully initialize. Wait for the log message: Started Application in X seconds
